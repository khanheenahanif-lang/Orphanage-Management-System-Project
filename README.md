# Orphanage Management System Project

The Orphanage Management System is a web-based application designed to streamline and digitize orphanage operations. It helps administrators efficiently manage child records, staff details, donations, adoptions, inventory, and daily activities through a centralized platform.

**Features:**
Child profile and record management
Staff and volunteer management
Donation management
Adoption process management
Inventory and resource management
Secure authentication 
Certificate Generation
Survey Module

**Technologies Used:**
Frontend: HTML, CSS, JavaScript 
Backend:  Python 
Database: SQLite

**Objective:**
The goal of this project is to improve operational efficiency, maintain organized records, and enhance transparency in orphanage administration through digital management solutions.
